# Java 实现入口

本仓库 Java 实现位于：

- `src/main/java`
- 对应测试：`src/test/java`

## 已有覆盖（Java）

- 线性结构：单链表、双链表、栈、队列
- 排序：冒泡、快排、插入、希尔、归并、堆排、基数排序
- 查找：二分、哈希
- 字符串：KMP
- 树：AVL、红黑树、哈夫曼、线索树
- 图：Dijkstra、Floyd、SPFA、Prim、Kruskal、拓扑、关键路径

## 运行说明

> 依赖 JDK + Maven 环境

```bash
mvn clean test
```

如果本机没有 Maven，可先安装后再执行。
